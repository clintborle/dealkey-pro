Questions

Things you might be wondering

How is this different from Honey?

Honey uses a central server that tracks your shopping behavior and replaces affiliate links. DealKey runs entirely in your browser using your own AI key. No servers, no tracking, no affiliate hijacking.

What does the AI actually cost per search?

About $0.001 per search — roughly a tenth of a penny. A $5 credit on any provider will last months of daily shopping. You only pay when DealKey actually searches.

Is my API key safe?

Your API key is stored locally in Chrome's secure storage and never leaves your browser. DealKey has no servers, no analytics, no telemetry. The code is open source — verify for yourself.

Why pay $19 when the free version exists?

The Pro version from the Chrome Web Store gives you automatic updates, one-click install, and email support. It also supports independent development. The AI engine is identical in both versions.

Which AI provider should I use?

We recommend Claude (Anthropic) for the best results, but OpenAI and OpenRouter both work well. OpenRouter gives you access to multiple models from one key if you like flexibility.

Does it work on every store?

DealKey works on any website with a checkout page. It detects coupon fields automatically and searches for valid codes using AI. Results vary by store — some have more active codes than others.

Can I see the source code?

Yes. DealKey is fully open source. Every line of code is on GitHub. The Pro version from the Chrome Web Store is the same code — you're paying for convenience and support, not a different product.

What data does DealKey collect?

None. Zero. DealKey has no servers, no analytics, no tracking pixels, no telemetry. Your browsing stays in your browser. The only network requests are from your browser to the AI provider you chose.
