Privacy Policy

Last updated: April 15, 2026

The short version: DealKey doesn't collect your data. Your API key, browsing activity, and purchase history never leave your browser. There are no DealKey servers processing your information.

What DealKey is

DealKey is a Chrome browser extension that uses an AI API key you provide to search for discount codes at checkout. The extension runs entirely within your browser. There are no DealKey servers involved in the code search process.

Data we do not collect

DealKey does not collect, transmit, store, or sell any of the following:

  • Your browsing history or activity
  • The websites you visit or shop on
  • Your purchase history or order values
  • Your AI API key
  • Your name, email address, or any personal information
  • Any analytics, usage data, or telemetry

What stays in your browser

The following data is stored locally in Chrome's secure storage on your device only. It is never transmitted to DealKey:

  • Your AI provider selection (Claude, OpenAI, or OpenRouter)
  • Your AI API key
  • Your DealKey license key (if you purchased Pro)
  • Cached discount code results for the current session

Third-party API calls

When you use DealKey, your browser makes direct API calls to the following services. These calls go directly from your browser to the provider — DealKey never sees or processes them:

  • Your AI provider (Anthropic, OpenAI, or OpenRouter) — to search for discount codes. The page URL is sent to generate relevant results. No personal data is included.
  • Gumroad — only when you validate a Pro license key. Your license key is sent to confirm it's valid. No other data is transmitted.

Permissions

DealKey requests the following Chrome permissions:

  • activeTab — to read the URL of the current checkout page so DealKey can search for relevant codes
  • storage — to save your API key and settings locally in Chrome
  • scripting — to detect checkout fields on the page and apply codes automatically

DealKey does not use these permissions to track your activity, read page content outside of checkout detection, or transmit any data to external servers.

Open source

DealKey's source code is publicly available on GitHub. You can verify exactly what the extension does and doesn't do.

Changes to this policy

If this policy changes, the updated date at the top of this page will reflect it. We will not retroactively change data practices — any changes will only apply going forward.
